# 👥 13. Team Details — BOT THINGS

The RescueBOT project was successfully developed through cross-disciplinary teamwork involving software engineering, hardware integration, robotics, communication systems, IoT connectivity, AI-based detection, and mechanical assembly. Each team member played a crucial role in contributing their expertise to build a smart, multi-hazard rescue robotic platform capable of operating in dangerous environments.

---

### 1. Prolayjit Biswas — Team Lead, Software & System Integration

| Field | Detail |
|:---|:---|
| **Role** | Team Lead, Software & IoT Developer |
| **GitHub** | [@Prolayjit-B14](https://github.com/Prolayjit-B14) |
| **Core Expertise** | Web Dashboard Development, C/C++ Programming, MQTT Integration, AI Models, RF Telemetry |

**Contributions:**
- Overall software architecture design and project coordination.
- Developed the RescueBOT website and web dashboard for real-time monitoring and remote control of the system.
- Integrated GPS tracking and connected location data from the GPS module to the website for live visualization.
- Implemented MQTT-based communication for fetching sensor data and enabling real-time updates between the rover and dashboard.
- Worked on ESP32-CAM integration to provide live camera feed access directly through the web interface.
- Contributed to AI-based detection systems including human detection and fire detection for intelligent disaster response.
- Managed firmware-to-website communication, remote communication technologies (LoRa and NRF), sensor data visualization, and overall software-system coordination.

┌─────────────────────────────────────────────────────────┐
